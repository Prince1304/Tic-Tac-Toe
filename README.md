# ❌ Tic-Tac-Toe ⭕

A clean, modern, and interactive web-based Tic-Tac-Toe game built using vanilla HTML, CSS, and JavaScript. Experience the classic game right in your browser with smooth animations and a responsive design.

🔗 **[Live Demo](https://prince1304.github.io/Tic-Tac-Toe/)**

---

## ✨ Features

- **Interactive Gameplay:** Classic 2-player grid turns ('X' and 'O') with real-time turn tracking.
- **Winner Detection:** Instantly highlights the winning combination and announces the winner.
- **Draw Indicator:** Smartly detects when the grid is full with no winner, declaring a draw.
- **Quick Reset:** A dedicated reset button to instantly clear the board and start a new match.
- **Fully Responsive:** Beautifully optimized for desktop, tablet, and mobile displays.

---

## 🛠️ Tech Stack

This project is built using pure frontend web technologies with no external frameworks, ensuring quick load times and high performance:

- **HTML5:** Structured semantic markup for the game grid and overlay modals.
- **CSS3:** Custom styles, modern flexbox/grid layout design, and smooth hover effects.
- **JavaScript (ES6):** Core game logic, state management, win/draw checking conditions, and DOM manipulation.

## 🚀 Getting Started

If you want to run this project locally or make changes to the code, follow these simple steps:

### Prerequisites
All you need is a modern web browser installed on your computer.

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/prince1304/Tic-Tac-Toe.git](https://github.com/prince1304/Tic-Tac-Toe.git)
   ```

2. Navigate to the project directory:
   ```bash
   cd Tic-Tac-Toe
   ```
3. Open the game:
   Simply double-click the index.html file to launch it in your default browser.
